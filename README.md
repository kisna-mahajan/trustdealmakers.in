# Trust DealMakers

**International Business Consulting** — trustdealmakers.in

A static website for Trust DealMakers, an international business consulting firm founded by Anil Mahajane. Hosted on GitHub Pages with Cloudflare in front.

---

## About Trust DealMakers

Trust DealMakers provides mandate-driven, execution-focused advisory across three service areas:

- **International Trade & Indenting** — Cross-border buyer/supplier identification, commercial negotiation, and trade facilitation
- **Industrial Asset Monetisation** — International redeployment of surplus plants and machinery during technology upgrades
- **International Representation & Market Development** — Connecting global manufacturers, buying houses, and technology companies with qualified customers in India

### Industries Served

Machinery manufacturers · Industrial chemicals · Specialty chemicals · Medical equipment · Hospital devices · Global buying houses · Procurement companies · Technology companies

## Tech Stack

- Plain HTML, CSS, JavaScript — no framework or build tool
- Hosted on **GitHub Pages** (`main` branch, root)
- DNS and CDN via **Cloudflare**
- Custom domain: `trustdealmakers.in`
- Font: [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- Design inspiration: [Bain & Company](https://www.bain.com)

---

## Deployment

The site deploys automatically when changes are pushed to `main`. No CI/CD configuration needed.

To make changes:
1. Edit `index.html`
2. Commit and push to `main`
3. GitHub Pages serves the update within ~30 seconds
